# Обзоры

Здесь собраны подробные обзоры и размышления о прослушанных аудиокнигах.

## Как написать обзор

1. Создайте новый файл в директории `docs/reviews/`
2. Используйте формат: `название-обзора.md`
3. Следуйте шаблону ниже:

```markdown
---
title: Обзор на "Название книги"
book: Название книги
author: Автор книги
date: YYYY-MM-DD
tags: [тег1, тег2, тег3]
rating: Рейтинг (1-5)
---

# {{ $frontmatter.title }}

**Книга:** {{ $frontmatter.book }}
**Автор:** {{ $frontmatter.author }}
**Дата обзора:** {{ $frontmatter.date }}
**Рейтинг:** {{ $frontmatter.rating }}/5
**Теги:** {{ $frontmatter.tags.join(', ') }}

## Общее впечатление

Ваше общее впечатление от книги...

## Что понравилось

- Пункт 1
- Пункт 2

## Что не понравилось

- Пункт 1
- Пункт 2

## Основные мысли

Глубокий анализ и размышления...

## Кому рекомендую

Для кого эта книга будет особенно интересна...

## Выводы

Итоговые мысли и рекомендации...
```
