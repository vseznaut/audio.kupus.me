## Описание

Уникальная книга, которая представляет теорию управления проектами в форме увлекательного романа. Главный герой получает задание руководить разработкой шести программных продуктов в вымышленной стране Моравия, применяя современные методы управления проектами.

## Основные темы

- Управление IT-проектами
- Риск-менеджмент
- Командная работа
- Реалистичное планирование
- Управление людьми

## Для кого эта книга

Книга будет полезна:
- Проектным менеджерам
- Руководителям IT-команд
- Разработчикам ПО
- Всем, кто занимается управлением проектами

## Ключевые выводы

Книга в захватывающей форме демонстрирует принципы эффективного управления проектами, показывая, как теория работает на практике. Обязательное чтение для каждого проектного менеджера.
