# Аудиокниги

Добро пожаловать в каталог аудиокниг! Здесь собраны все прослушанные аудиокниги с описаниями и метаданными.

## Как добавить новую аудиокнигу

1. Создайте новый файл в директории `docs/audiobooks/`
2. Используйте формат: `название-книги.md`
3. Следуйте шаблону ниже:

```markdown
---
title: Название книги
author: Автор
narrator: Чтец
duration: Продолжительность (часы)
genre: Жанр
rating: Рейтинг (1-5)
finished: Дата окончания (YYYY-MM-DD)
---

# {{ $frontmatter.title }}

**Автор:** {{ $frontmatter.author }}
**Чтец:** {{ $frontmatter.narrator }}
**Жанр:** {{ $frontmatter.genre }}
**Продолжительность:** {{ $frontmatter.duration }}
**Рейтинг:** {{ $frontmatter.rating }}/5
**Прослушано:** {{ $frontmatter.finished }}

## Описание

Краткое описание книги...

## Впечатления

Ваши впечатления от прослушивания...

## Цитаты

> Интересная цитата из книги

## Ссылки

- [Где послушать/купить](#)
```
